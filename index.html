⚡ Awsome-QuantManus-SKILLS — Agent 技能生态
为 QuantManus 构建的可插拔技能扩展体系。每个 Skill 是一份结构化 SKILL.md 文档，Agent 在规划和执行任务时按需加载，形成完整的量化 AI Agent 生态。 独立仓库维护，社区可贡献。
🏗️ Skills 架构机制

全局安装于 ~/.quantmanus/skills/，每个 Skill 为独立子目录，包含 SKILL.md（Frontmatter 元数据 + Markdown 正文）。支持 Always-on（注入 System Prompt）与按需加载两种模式。

🔌 依赖与发现

Skill 通过 requires 字段声明 CLI 工具和环境变量依赖，Agent 启动时 SkillsLoader 自动扫描、检查依赖、构建技能列表，以 XML 摘要注入 System Prompt 供 Planner 决策。

🧩 模块化分层

Skills（全局技能）与 Bootstrap（项目级自定义）、Memory（项目级记忆）三层分离。技能安装一次即可跨项目复用，项目配置与记忆跟随工作目录。

📦 社区可扩展

独立 GitHub 仓库（Awsome-QuantManus-SKILLS）维护技能组，任何人可贡献新 Skill，只需创建目录并编写 SKILL.md。

Agent 启动 → SkillsLoader 扫描 skills/ → 解析 Frontmatter → 检查 requires 依赖
  ├─ Always-on 技能 → 全文注入 # Active Skills
  ├─ 所有技能 → XML 摘要注入 # Skills（name / description / available）
  └─ 运行时 → Planner 参考摘要规划 → 按需 read_file 加载完整 SKILL.md
